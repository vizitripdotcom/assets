# Assets Vizitrip

Assets Vizitrip adalah repository yang berisi file-file yang diperlukan untuk mengembangkan aplikasi Vizitrip.

## Struktur Folder

- `css` : folder yang berisi file-file CSS yang diperlukan untuk mengembangkan aplikasi Vizitrip.
- `js` : folder yang berisi file-file JavaScript yang diperlukan untuk mengembangkan aplikasi Vizitrip.
- `img` : folder yang berisi file-file gambar yang diperlukan untuk mengembangkan aplikasi Vizitrip.
- `fonts` : folder yang berisi file-file font yang diperlukan untuk mengembangkan aplikasi Vizitrip.
- `scss` : folder yang berisi file-file SCSS yang diperlukan untuk mengembangkan aplikasi Vizitrip.
- `vendor` : folder yang berisi file-file vendor yang diperlukan untuk mengembangkan aplikasi Vizitrip.
